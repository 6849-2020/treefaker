Welcome to TreeFaker, an online, lightweight version of Robert Lang's TreeMaker. To get started, first draw a tree in the box above. Ctrl + drag a vertex to add a new leaf to the tree. Shift + click on an edge to delete it. Right click on an edge to manually set its length. When you are done drawing the tree, click the "Generate Disk Packing" button.

Once you have found a satisfactory disk packing, click the "Get Crease Pattern" button to show the crease pattern.
